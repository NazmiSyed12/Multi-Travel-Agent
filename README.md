# ✈️ Multi-Agent Travel Planning System

**My name is NAZMI SYED, a Dual Degree student at IIT Kharagpur, and I built this project to redefine travel planning.**

I created the Multi-Agent Travel Planning System using a powerful multi-agent AI system that transforms your preferences into a personalized, detailed itinerary — in minutes, not hours.

---

## 🌍 Overview

I wanted to say goodbye to travel planning stress. 
I built this system to simplify the entire process by letting you describe your dream trip once. From there, intelligent agents handle everything — flights, stays, activities, costs, and logistics — and deliver a seamless, day-by-day itinerary tailored just for you.

---

## ⚙️ How It Works

1. ** Tell Us Your Plans**
   Fill out a simple form with your destination, travel dates, budget, travel style, and preferences.

2. ** Let AI Agents Work**
   Behind the scenes, I programmed specialized AI agents to collaborate and plan flights, accommodations, activities, routes, and budgeting.

3. ** Receive Your Itinerary**
   Get a personalized travel plan with:

   - Suggested flights and hotels
   - Curated activities for each day
   - Estimated costs and booking links

## 🛠️ Tech Stack I Used

### Frontend

- [Next.js](https://nextjs.org/) – Fast, modern React framework
- [React](https://reactjs.org/) – Interactive UI components
- [TypeScript](https://www.typescriptlang.org/) – Type-safe JavaScript

### Backend

- [FastAPI](https://fastapi.tiangolo.com/) – High-performance Python API
- [Python](https://www.python.org/) – Backend logic & AI orchestration
- [PostgreSQL](https://www.postgresql.org/) – Robust relational database I chose for robust data storage.

### AI & Integrations

- **[Agno](https://www.agno.com/)** – AI agent coordination and memory
- **[Exa](https://exa.ai/)** – Context-aware search functionality
- **[Firecrawl](https://www.firecrawl.dev/)** – Web scraping and data extraction
- **[Gemini](https://deepmind.google/technologies/gemini/)** – Core LLM I integrated for planning and reasoning
- **External APIs** – Real-time data from Google Flights, Kayak, and more
